## System Requirements

[Basic Requirements]

- C++ compilator Version g++ 15.1.1 or later
- Qt Version 6 or later
- Ninja 1.12.1 or later
- CMake Version 3.20 or later
- Git Version 2.49.0 or later
- Clang-format Version 19.1.7 or later
- yt-dlp 2025.05.22 or later
